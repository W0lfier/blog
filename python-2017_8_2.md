- `__str__`:给类添加`print`功能
- `__repr__`:给类添加==直接输出==功能
	
	```
	In [1]: class test(object):
	   ...:     def __init__(self,name):
	   ...:         self.name = name
	   ...:     def __str__(self):
	   ...:         return 'Name is %s.' % self.nam
	   ...:     __repr__ = __str__
	   ...:

	In [2]: student = test('jack')

	In [3]: student
	Out[3]: Name is jack.

	In [4]: print(student)
	Name is jack.
	```
